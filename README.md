# analisis-ventas-bi-sql
Implementación de un sistema de Business Intelligence con SQL Server y Power BI.
## Descripción del Proyecto
Este proyecto simula la implementación de un sistema de BI para una multinacional de retail. El objetivo fue transformar datos crudos de ventas en insights estratégicos para mejorar la toma de decisiones.

## Tecnologías utilizadas
* **SQL Server (SSMS):** Modelado multidimensional y gestión de base de datos relacional.
* **Power BI:** Diseño de dashboards interactivos y visualización de KPIs.
* **Google Sheets:** Limpieza inicial y preprocesamiento de datos.

## Logros Principales
* **Relevamiento Funcional:** Definición de requerimientos de negocio y KPIs críticos (Ventas por territorio, rentabilidad por categoría).
* **Modelado de Datos:** Creación de un esquema eficiente para soportar el análisis de más de 10,000 registros.
* **Análisis Exploratorio (EDA):** Identificación de patrones estacionales y productos de mayor impacto en la recaudación.
